\chapter{The Athena Instruction Set}

\section{Instruction Format}

There are 4 basic Athena instruction formats:
\begin{itemize}
    \item R-Format
    \item I-Format
    \item J-Format
    \item LS-Format
\end{itemize}

\subsection{R-Format}

\Rinstr{opcode}{ra}{rb}{rc}{shmat}{func}

\subsection{I-Format}

\Iinstr{opcode}{ra}{rb}{imm}

\subsection{J-Format}

\Jinstr{opcode}{ra}{offset}

\subsection{L/S-Format}

\LSinstr{opcode}{ra}{rb}{\rotatebox{90}{width}}{offset}
